# ALOvchinnikov_memorial

Сайт мемориальной выставки «Пространство памяти», посвящённой художественному наследию Л.А. и А.Л. Овчинниковых.

## Структура

- `index.html` — разметка сайта и SPA-страниц
- `styles.css` — стили сайта
- `app.js` — роутинг, анимации, каталог работ и логика 3D-мастерской

## Текущий стек

Статический сайт на HTML/CSS/JS с hash-routing и Three.js для 3D-мастерской.

## Публикация

Для GitHub Pages достаточно держать `index.html` в корне репозитория.
